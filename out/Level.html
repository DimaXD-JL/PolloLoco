Class: Level

Level(enemies, clouds, coins, bottles, backgroundObjects)

Represents a game level containing all game objects and environment elements. This class serves as a container for organizing all entities within a specific game level.

Constructor

new Level(enemies, clouds, coins, bottles, backgroundObjects)

Creates a new Level instance with specified game objects.
Parameters:

clouds Array.<Cloud> Cloud objects for the level's sky

backgroundObjects Array.<BackgroundObject> Background elements for the level
